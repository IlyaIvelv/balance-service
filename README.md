# Balance Service

REST API сервис для управления пользовательскими счетами с автоматическим начислением процентов и переводами.

## 🛠 Технологический стек

| Категория | Технологии |
|-----------|------------|
| **Язык** | Java 11 |
| **Фреймворк** | Spring Boot 2.7.18 |
| **БД** | PostgreSQL 13 |
| **Кэш** | Redis 7 |
| **ORM** | jOOQ 3.16.18 |
| **Миграции** | Liquibase 4.9.1 |
| **Безопасность** | Spring Security + JWT |
| **Сборка** | Maven |
| **Контейнеризация** | Docker, Docker Compose |
| **Документация** | Swagger UI (SpringDoc) |

## 🚀 Быстрый старт

### 1. Подготовка

Убедитесь, что установлены:
- Java 11+
- Maven 3.6+
- Docker & Docker Compose

### 2. Запуск инфраструктуры (БД + Redis)

```bash
docker-compose up -d
```
Это запустит PostgreSQL на порту 5432 и Redis на порту 6379.

### 3. Запуск приложения
###    Вариант А: Через IDEA
   Просто запустите класс com.balanceservice.Main.
###    Вариант Б: Через Maven
```bash
mvn spring-boot:run
```
### 4. Проверка
   Откройте Swagger UI в браузере:
   👉 http://localhost:8080/swagger-ui.html

### 🔐 API Документация
## Аутентификация
Для работы с защищенными методами нужен JWT токен.
1. POST /api/v1/users/register — Регистрация (открытый эндпоинт)
2. POST /api/v1/auth/login — Логин (возвращает токен)
3. В Swagger нажмите кнопку Authorize и вставьте токен
## Основные эндпоинты
* GET /api/v1/users — Поиск пользователей (фильтры + пагинация)
* PUT /api/v1/users/me/contacts — Обновление контактов текущего юзера
* POST /api/v1/transfers — Перевод денег (с валидацией лимитов)

### 💡 Бизнес-логика
## 1. Начисление процентов
   Каждые 30 секунд запускается задача (Scheduler), которая:
   Увеличивает баланс на +10%
   Ограничивает рост: balance не может превышать initial_balance * 2.07 (207%)
## 2. Переводы
   При переводе проверяется:
   Достаточно ли средств у отправителя
   Не превысит ли баланс получателя лимит в 207% от его начального депозита
   Запрещены переводы самому себе
